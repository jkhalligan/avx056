Accessibility Statement for InheritNOW<\/span> website<\/h2>\n\n\n\n

Conformance status<\/h3>\n\n\n\n

The Web Content Accessibility Guidelines (WCAG)<\/a> defines requirements for designers and developers to improve accessibility for people with disabilities. It defines three levels of conformance: Level A, Level AA, and Level AAA. InheritNOW<\/span> website is fully conformant with WCAG 2.0 level AA. Fully conformant means that the content fully conforms to the accessibility standard without any known exceptions.<\/p>\n\n\n\n

Feedback<\/h3>\n\n\n\n

We welcome your feedback on the accessibility of InheritNOW<\/span> website. Please let us know if you encounter accessibility barriers on InheritNOW<\/span> website:<\/p>\n\n\n\n

  • E-mail: accessibility@inheritnow.com<\/a><\/li>
  • Postal Address: 1002 Justison Street, Wilmington, DE 19801<\/li><\/ul>\n\n\n\n

    We try to respond to feedback within ten (10) business days.<\/p>\n\n\n\n

    Technical specifications<\/h3>\n\n\n\n

    Accessibility of InheritNOW<\/span> website relies on the following technologies to work with the particular combination of web browser and any assistive technologies or plugins installed on your computer:<\/p>\n\n\n\n

    • HTML<\/li>
    • CSS<\/li>
    • JavaScript<\/li><\/ul>\n\n\n\n
